# Savera Handicrafts — Interactive Catalogue & Pricing System

A complete browser-based catalogue and pricing tool for Savera Handicrafts. No installation or server required — just open `index.html`.

## Features
- Interactive product catalogue with category filters
- Dynamic pricing with currency/FX support
- Export functionality (Excel/XLSX)
- Search and sort across all products
- Clean gold & ivory brand design

## How to Run
1. Clone or download this repository
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox)
3. That's it — fully offline, no dependencies to install
